Standard grounds

Tyle lets engineers model their "objects" with the established XML format. Because the format is very generic, it will easily adapt to the wildest data structures.

As data lives in plain text files, it can be stored and accessed with simple shared-folders, FTP, or even Dropbox.

No mysterious database technology is involved. No complex infrastructure is needed: A text editor is enough if you have to change a number in there.

<wheel>
  <diameter>50</diameter>
  <width>10</width>
  <invented-on>5000BC</invented-on>
</wheel>

> Simple, I get it

Data augmentation

The next step is to augment the model with any number of built-in or custom views.

<wheel views="shapes.cylinder">
  <diameter views="shapes.cylinder.d">50</diameter>
  <width views="shapes.cylinder.w">10</width>
  <invented-on>5000BC</invented-on>
</wheel>

Here, we are creating a view to display the primitive shape of a wheel in 3D.

But views can also consist of text, tables, graphs, schematics...

Oh! And you can display one single object, or compare multiple objects with each other.

> Looks good

Import/Export

Insert Tyle into existing workflows, by augmenting your data with importers and exporters.

As their names suggest, they allow you to map Tyle's data format in and out of your existing databases and tools.

Write a custom view, importer or exporter with Javascript, the language of the web.

Leverage any library available out there simply by declaring it as a dependency. Check-out d3.js or three.js for inspiration.

Tyle-Explorer

Access your object's views, compare various objects and run importers & exporters.

Tyle-Explorer is available in the browser (if your data is stored on a server) or as a standalone application (to access data locally).

Version-controlled data

Need to keep track of evolutions in your data in time?

Need to access/edit data concurrently?

Need to maintain parallel versions of the data?

Tyle will work very nicely with tools like Git, Subversion, or even Dropbox!

Git does a great job at tracking changes in your data, and merging the work done by various engineers concurrently.

Bonus: Combine Git and GitHub, to instantly share your data on the web. Then, host Tyle-Explorer on a static page to displays your views to world!
